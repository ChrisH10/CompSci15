HTML

HTML stands for HyperText Markup Language.

HTML was created in 1993 and is currently on it's fifth release, HTML 5.0. It was developed by W3C & WHATWG.

This is how HTML 5.0 is setup:

  • "Head" stands for Header
  • "Body" stands for Body of paragraph
  • "P" stands for Paragraph
  • HTML tag inside the code stands for where the text is assigned to be HTML code.

    • What are Tags & Attributes?

    Tags are elements that are used to define a type of code that is placed between the tags. For instance, "P" tag is used to define that there is a paragraph of text placed in between the element and should be treated as a plain text file.

    Attributes are used in HTML to sort similar elements together. For instance, the "Title" attribute is used to give context to all of the elements in the HTML code.

    Tags & Attributes Examples:

    3. "abbr" Defines an abbreviation
