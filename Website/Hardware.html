Motherboard

The motherboard holds all the transistors and connections that connect everything together. It contains the BIOS and the CMOS battery.

CPU

CPU is the Central Processing Unit. It does all the calculations for the computer. It requires a fan to keep cool and the BIOS is tied to the CPU

RAM

RAM is Random Access Memory. Random Access Memory allows for the computer to store temporary calculations that will be later overritten when a new calculation is needed to be calculated.

Hard Drive

A Hard Drive stores all of your information that you save on your computer. It was first introduced in 1956.

Power Supply

A Power Supply is used to power all of the components of the computer.

Communication

These components abvoe communicate with eachother through the motherboard. The components are given calculations through the motherboard.

Keyboard,Mouse and Display

The keyboard and mouse are called input devices. These input devices are used to give commands to each component. The display is an output device. It displays the calculations that you input with the keyboard and mouse.
